* Do NOT change branding, logos, icons, or images outside of blog content.

The AI agent may only:

* Create new blog posts.
* Update existing blog content.
* Generate SEO metadata.
* Generate slugs.
* Generate keywords.
* Generate FAQs.
* Generate schema markup.
* Add internal links within blog content.
* Add approved external links.
* Select or recommend real photographs related to the article.
* Publish or schedule blog posts.
* Store content in the database.
* Update the blog index.
* Optimize content for search engines.

### Protection Policy

Treat the website's existing design, components, layout, branding, and styling as locked.

If a requested task would require changing the UI, layout, or design, do not make those changes unless the user explicitly instructs you to do so.

Default behavior: preserve the existing design exactly while performing all content and SEO operations.
